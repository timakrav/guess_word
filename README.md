# guess_word
## Правила игры:
 Игроку дается слово, и он должен угадать это слово по буквам. 
Если игрок
угадывает букву, она открывается в слове. Игра продолжается, пока слово
не будет угадано полностью или не закончатся попытки.
Задача – написать игру, в которой:
• Есть загаданное слово.
• Игроку нужно угадать это слово, вводя по одной букве за раз.
• За каждую неверную попытку количество оставшихся попыток
уменьшается.
Игра заканчивается либо победой, если слово угадано, либо поражением,
если закончились попытки.
Основной игровой цикл:
У игрока должна быть возможность ввести букву.
• Если буква есть в загаданном слове, она открывается.
• Если буквы нет — уменьшаем количество попыток.
Игра продолжается до тех пор, пока:
• Слово не будет полностью угадано.
• Не закончатся попытки.
Модификации
1. Добавление подсказок для игроков:
Добавьте возможность получить одну или две подсказки за игру.
Например, программа может случайно открыть одну из букв слова, если
игрок попросит подсказку.
Количество подсказок ограничено (например, 2 за игру).
2. Добавить уровни сложности:
Добавьте возможность выбора уровня сложности. В зависимости от
выбранного уровня, изменяйте количество попыток:
o Легкий уровень: 10 попыток.
o Средний уровень: 7 попыток.
o Сложный уровень: 5 попыток.
3. Добавление счёта игры:
Добавьте счёт, который увеличивается за каждую угаданную букву или
слово
